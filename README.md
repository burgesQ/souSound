# souSound

A privat esound sharing project. Nov 2017

Process to deploy the project :

```bash
$ ./scrit/IpDocker
# copy/ast db ip on .env
$ ./scrit/genDb
```

The project is based on the latest sensioLab php framework : Symfony Flex.
